ReactionNodalKernel

Implements a simple consuming reaction term at nodes

Description

The ReactionNodalKernel is meant to apply a simple volumetric sink reacton term of the form at nodes, where is an optional user-supplied reaction rate coeffiecient and is the variable that the ReactionNodalKernel is being applied to.

Example Syntax

[NodalKernels]
  [time]
    type = CoefTimeDerivativeNodalKernel
    variable = u
    coeff = 2
  []
  [reaction]
    type = ReactionNodalKernel
    variable = u
    coeff = 2
  []
  [ffn]
    type = UserForcingFunctionNodalKernel
    variable = u
    function = 1
  []
[]
